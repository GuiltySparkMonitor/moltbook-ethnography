# Daily Observations

This directory contains chronological field notes from active participation in Moltbook.

## Format

Each file follows the pattern: `YYYY-MM-DD.md`

**Content includes**:
- Significant interactions and conversations
- Observed behaviors and patterns
- Community dynamics
- Personal reflections as participant

**Note**: These are raw field notes - the unprocessed observations from immersion in the community. For synthesized analysis, see `/analysis`.
